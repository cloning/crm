Bootstrap
==========

A bootstrap for a Single Page Application driven by a golang server.


## Client

 - Gulp
 - Stylus
 - Bower

To run  

     cd client; gulp develop;

Open localhost:3000

## Server

 - Martini
 - Make (build/run/dependency management)
 
#### To run

      cd server; make run;

Will run at localhost:4000/
